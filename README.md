# router
Bus routing app using NextBus API

## Instructions

- Clone the project: `git clone https://github.com/menan/router.git`
- Navigate the project folder: `cd router`
- Install necessary pods: `pod install`
- Open *Router.xcodeworkspace*
- Run the project :)
